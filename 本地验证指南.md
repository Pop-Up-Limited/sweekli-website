# 本地验证 Contentful 集成指南

## 🚀 快速验证步骤

### 1. 启动开发服务器

```bash
npm run dev
```

服务器会在 `http://localhost:5173` 启动

### 2. 访问 Insights 页面

打开浏览器访问：
- **英文版**：http://localhost:5173/insights
- **中文版**：http://localhost:5173/zh/insights

### 3. 检查内容

✅ **应该看到**：
- 从 Contentful 获取的文章列表
- 文章标题、摘要、日期、分类
- 封面图片（如果已上传）

❌ **如果看到**：
- "Loading articles..." - 正在加载
- "No articles found." - Contentful 中没有文章，需要先创建
- 错误信息 - 检查浏览器控制台

### 4. 测试多语言切换

1. 在网站右上角切换语言
2. 文章内容应该相应切换（标题、摘要、分类）
3. 日期格式也会相应变化

### 5. 检查首页

访问：http://localhost:5173/

滚动到 **Insights Section**，应该显示最新 3 篇文章。

---

## 🔍 调试方法

### 打开浏览器开发者工具

1. 按 `F12` 或 `Cmd+Option+I`（Mac）
2. 切换到 **Console** 标签
3. 查看是否有错误信息

### 常见错误

#### 错误 1：`401 Unauthorized`
- **原因**：Access Token 错误或过期
- **解决**：检查 `src/utils/contentful.ts` 中的 ACCESS_TOKEN

#### 错误 2：`404 Not Found`
- **原因**：Content Type ID 错误或 Space ID 错误
- **解决**：检查 Content Type ID 是否为 `blogPost`

#### 错误 3：`No articles found`
- **原因**：Contentful 中没有已发布的文章
- **解决**：在 Contentful 后台创建并发布文章
