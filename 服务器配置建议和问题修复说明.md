# 服务器配置建议和问题修复说明

## 一、腾讯云CVM服务器配置建议

### 基础配置
- **CPU**: 2核
- **内存**: 4GB
- **硬盘**: 50GB SSD云硬盘
- **带宽**: 建议选择5Mbps（可根据实际访问量调整）

### 操作系统选择
推荐使用以下操作系统之一：

1. **Ubuntu 22.04 LTS**（推荐）
   - 版本：Ubuntu Server 22.04 LTS 64位
   - 优势：长期支持版本，社区活跃，文档丰富
   - 适合：大多数Web应用场景

2. **CentOS 7.9**（备选）
   - 版本：CentOS 7.9 64位
   - 优势：稳定可靠，企业级应用广泛使用
   - 注意：CentOS 8已停止维护，建议使用CentOS 7.9

3. **Debian 11**（备选）
   - 版本：Debian 11 (Bullseye) 64位
   - 优势：轻量级，稳定性好

### 地域选择
建议选择：
- **华南地区（广州）**：如果主要用户在中国南方
- **华东地区（上海）**：如果主要用户在中国东部
- **华北地区（北京）**：如果主要用户在中国北方

### 网络配置
- **公网IP**: 选择"分配公网IP"
- **带宽计费模式**: 按流量计费（适合初期）或按固定带宽计费（适合稳定访问量）
- **安全组**: 需要开放以下端口：
  - 22（SSH）
  - 80（HTTP）
  - 443（HTTPS）

### 购买步骤
1. 登录腾讯云控制台
2. 进入"云服务器CVM" -> "实例"
3. 点击"新建"按钮
4. 选择计费模式（推荐：包年包月，更经济）
5. 选择地域和可用区
6. 选择实例配置（2核4G）
7. 选择镜像（推荐：Ubuntu 22.04 LTS）
8. 选择存储（50GB SSD云硬盘）
9. 配置网络（选择VPC和子网，分配公网IP，选择5Mbps带宽）
10. 配置安全组（开放22、80、443端口）
11. 设置主机名和密码
12. 确认订单并支付

### 备案准备
购买服务器后，需要准备以下材料进行备案：
- 域名证书
- 企业营业执照
- 法人身份证
- 网站负责人身份证
- 备案授权书（如需要）

备案流程通常需要7-20个工作日，建议尽快开始备案流程。

## 二、正品查询页面功能修复

### 问题描述
1. 手机版扫描功能：能调出摄像头，但无法扫描二维码
2. 手动输入验证码：无论输入什么，都提示"Invalid Code"

### 修复内容

#### 1. API接口配置
- **API地址**: `https://cha12315.com/api/wiki.html`
- **请求方法**: POST
- **请求格式**: JSON
- **请求体**: `{"code": "验证码"}`

#### 2. 响应处理
- API返回格式：`{ "authentic": true/false, ... }`
- 根据`authentic`字段判断验证结果

#### 3. 扫描功能
- 使用`html5-qrcode`库实现二维码扫描
- 支持移动设备摄像头访问
- 自动识别二维码并填充验证码

#### 4. 测试建议
建议使用以下测试验证码进行测试（需要从API提供方获取正确的测试码）：
- 正确验证码：用于测试成功场景
- 错误验证码：用于测试失败场景
- 空验证码：用于测试边界情况

## 四、技术实现说明

### 1. 语言切换数字显示修复
- 问题原因：语言切换时，动画计数器没有立即更新为新的数值
- 解决方案：在语言切换时，立即将显示值更新为当前语言对应的最终值，然后重新启动动画

### 2. 正品查询API修复
- 问题原因：API请求格式和响应处理不正确
- 解决方案：
  - 修改请求格式为JSON：`{"code": "验证码"}`
  - 修改响应处理逻辑，检查`authentic`字段
  - 使用`html5-qrcode`库实现二维码扫描功能

### 3. 微信二维码显示修复
- 移动端：点击微信图标后显示全屏弹窗，包含二维码图片
- 桌面端：鼠标悬停在微信图标上时显示二维码弹窗

## 五、后续建议

1. **服务器部署**
   - 建议使用Nginx作为反向代理
   - 配置SSL证书（Let's Encrypt免费证书）
   - 设置自动备份策略

2. **性能优化**
   - 启用CDN加速静态资源
   - 配置Gzip压缩
   - 优化图片大小和格式

3. **安全加固**
   - 定期更新系统和软件包
   - 配置防火墙规则
   - 启用DDoS防护

4. **监控告警**
   - 配置服务器监控
   - 设置告警规则
   - 定期检查日志
