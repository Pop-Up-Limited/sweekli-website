# 博客管理平台推荐

## 推荐平台对比

### 1. **Contentful** ⭐⭐⭐⭐⭐（最推荐）

**优点：**
- Headless CMS，专门为 API 设计
- 免费版：25,000 条内容记录/月，完全够用
- API 非常完善，文档清晰
- 界面简洁，客户容易上手
- 支持多语言、图片管理
- 有 Webhook，可以实时同步

**缺点：**
- 免费版有 API 调用限制（但通常够用）

**适合场景：**
- 需要简单易用的后台
- 客户不需要学习复杂系统
- 网站通过 API 读取内容

**官网：** https://www.contentful.com/

---

### 2. **Sanity** ⭐⭐⭐⭐

**优点：**
- 免费版：10,000 条文档，3 个用户
- 界面非常现代化，用户体验好
- API 强大，支持实时数据
- 可以自定义内容结构
- 有图片 CDN

**缺点：**
- 免费版用户数有限制

**适合场景：**
- 需要更灵活的内容结构
- 客户对界面要求较高

**官网：** https://www.sanity.io/

---

### 3. **Ghost** ⭐⭐⭐

**优点：**
- 专门为博客设计
- 界面简洁美观
- 免费开源（需要自己部署）
- 或者使用 Ghost Pro（付费托管）

**缺点：**
- 主要是博客功能，其他功能较少
- 如果自己部署需要服务器

**适合场景：**
- 只需要博客功能
- 不介意付费托管或自己部署

**官网：** https://ghost.org/

---

### 4. **Notion** ⭐⭐⭐⭐

**优点：**
- 完全免费
- 界面非常友好，客户容易上手
- 可以当数据库用
- 支持 API（需要 Notion API）

**缺点：**
- API 相对复杂一些
- 不是专门为 CMS 设计

**适合场景：**
- 预算有限
- 客户已经熟悉 Notion

**官网：** https://www.notion.so/

---

## 我的推荐

### 首选：**Contentful**

**理由：**
1. **简单易用**：客户在 Contentful 后台写文章，就像用 Word
2. **API 完善**：网站通过 API 读取，非常稳定
3. **免费够用**：25,000 条记录/月，对博客完全够用
4. **专业可靠**：很多大公司都在用
5. **图片管理**：内置图片 CDN，不需要额外配置

### 工作流程：

1. **你（开发者）**：
   - 注册 Contentful 账号
   - 创建 Space（工作空间）
   - 创建 Content Model（定义文章字段：标题、内容、日期等）
   - 获取 Space ID 和 Access Token
   - 在网站代码中接入 Contentful API

2. **客户**：
   - 登录 Contentful 后台
   - 创建新文章（就像写 Word 文档）
   - 发布后，网站自动显示最新文章

3. **网站**：
   - 通过 Contentful API 读取文章列表
   - 展示在 Insights 页面

## Contentful 接入步骤

### 1. 注册和创建 Space

1. 访问 https://www.contentful.com/
2. 注册账号（免费）
3. 创建新 Space（选择 "Empty space"）
4. 选择免费计划

### 2. 创建 Content Model

1. 进入 "Content model" 标签
2. 点击 "Add content type"
3. 创建 "Blog Post" 类型
4. 添加字段：
   - `title` (Short text) - 标题
   - `slug` (Short text) - URL 标识
   - `excerpt` (Long text) - 摘要
   - `content` (Rich text) - 正文
   - `image` (Media) - 封面图
   - `date` (Date & time) - 发布日期
   - `category` (Short text) - 分类

### 3. 获取 API 凭证

1. 进入 "Settings" > "API keys"
2. 复制：
   - **Space ID**
   - **Content Delivery API - access token**

### 4. 在网站中接入

我会帮你写好代码，你只需要：
- 把 Space ID 和 Access Token 填入环境变量
- 网站自动读取并展示文章

| Ghost | 开源（需自部署） | $9/月（托管） |

**结论：** Contentful 免费版对博客完全够用，除非你每天发布 800+ 篇文章。
