## 问题回答

### 1. 服务器选择问题

**问题**：中文网站需要备案，因为希望在微信内能访问网站，如果不备案的话会弹出网站真实性提醒。所以，中英文网站是否需要分开托管到不同服务器？

**回答**：
- **重要说明**：根据您的基础需求，**希望合并域名**（sweekli.co 与 sweekli.cn），因此**不建议分开托管**
- **推荐方案**：使用单一服务器架构
  - **统一托管**：中英文网站使用同一套代码，通过语言切换功能实现
  - **服务器选择**：
    - **方案A（推荐）**：使用境内服务器（如腾讯云国内节点）
      - 优点：可以备案，确保微信内正常访问
      - 缺点：海外访问可能较慢（可通过CDN优化）
      - 需要备案域名（sweekli.co 或 sweekli.cn）
    - **方案B**：使用境外服务器（如腾讯云香港节点）
      - 优点：海外访问速度快，无需备案
      - 缺点：微信内可能弹出真实性提醒
      - 可通过CDN加速国内访问
- **域名配置**：
  - 统一使用 `www.sweekli.com` 或 `www.sweekli.co` 或 `www.sweekli.cn`
  - 通过URL参数（`?lang=zh`）或路径前缀（`/zh/`）区分语言
  - 或使用子域名：`cn.sweekli.com`（中文）和 `en.sweekli.com`（英文），但指向同一服务器
- **备案建议**：
  - 如需在微信内正常访问，建议使用境内服务器并备案
  - 备案流程：使用腾讯云备案系统，准备营业执照、法人身份证等材料，备案周期通常为7-20个工作日
- **CDN配置**：
  - 无论选择哪种方案，都建议配置CDN加速
  - 境内服务器：可使用腾讯云CDN
  - 境外服务器：可使用CloudFlare等CDN服务

### 3. Contentful配置

**问题**：管理员的邮箱请设置为：zshi@sweekli.com，另外我想问下，这个支持开多个员工的账户进行内容上传吗？

**回答**：
- **管理员邮箱设置**：
  - ✅ 已设置为 zshi@sweekli.com
  - 如需修改，可在Contentful后台：Settings → Users → 找到管理员账户 → 编辑邮箱
- **多账户支持**：
  - **重要说明**：Contentful的免费套餐（Community）仅支持**1个用户**
  - **多人管理需要付费**：
    - **Team套餐**（$300/月起）：支持最多5个用户
    - **Professional套餐**（$300/月起）：支持最多10个用户
    - **Enterprise套餐**：支持更多用户，需联系销售
  - **当前状态**：
    - 网站使用Contentful的免费套餐
    - 如需多人管理，需要贵司自行升级到付费套餐
    - 升级后可以创建多个用户账户，每个账户可以有不同的权限级别：
      - **Admin**：完全管理权限
      - **Editor**：可以创建和编辑内容
      - **Author**：可以创建内容但需要审核
      - **Translator**：只能编辑翻译内容
  - **创建新用户**（升级后）：
    - 路径：Settings → Users → Invite users
    - 输入邮箱地址，选择角色，发送邀请
  - **建议**：
    - 如需多人管理，请先升级Contentful套餐
    - 为设计部同事创建Editor或Author角色
    - 为内容团队创建相应的角色
    - 定期审查用户权限，确保安全性

### 4. 中文版表单问题

**问题**：中文版国内的表单通过Tally是可以正常访问提交的吗？如果不行的话，是否有替代方案？

**回答**：
- **Tally在国内的访问问题**：
  - ⚠️ **重要**：Tally是国外服务，在中国大陆地区访问**可能不稳定或被限制**
  - Tally的服务器位于海外，国内用户访问可能遇到：
    - 加载速度慢
    - 偶尔无法访问
    - 提交失败的情况
- **解决方案**：
  - ✅ **已确认**：将切换为可直接嵌入网站的表单服务
  - **推荐方案**：使用**金数据（jinshuju.com）**或**问卷星企业版**
    - 这两个服务都支持直接嵌入到网站中（通过iframe或API方式）
    - 国内访问稳定，速度快
    - 支持自定义样式，可以完美融入网站设计
    - 数据提交后可直接发送到指定邮箱（zshi@sweekli.com）
    - 与腾讯问卷不同，这些服务提供完整的嵌入功能，不仅仅是链接跳转
  - **实施方式**：
    - 在表单服务中创建表单
    - 获取嵌入代码（iframe或JavaScript代码）
    - 集成到Contact页面中，替换现有的Tally表单
    - 表单将无缝显示在网站中，用户无需跳转到外部页面
