# Sweekli 网站使用说明

![Sweekli 网站首页](docs/images/sweekli-homepage.png)

## 🎨 网站设计亮点

### 首页的三个悬浮球动画

当你第一次打开网站首页时，会看到三个彩色的球（蓝色、红色、黄色）从屏幕外飞入，然后随机停在不同位置。这个设计有几个有趣的特点：

- **每次都不一样**：每次刷新页面或重新进入，这三个球的位置、大小和移动路径都是随机生成的，所以你永远不会看到完全相同的布局
- **动态效果**：球会先快速飞入，然后缓慢浮动，营造出轻盈、活泼的视觉感受
- **响应式设计**：在手机端，球的大小会自动缩小，确保在不同设备上都有良好的体验

这个设计灵感来源于"米散"的概念，象征着品牌在市场上的灵活性和多样性。

## 🌐 中英文切换功能

网站支持完整的中英文双语切换：

- **切换按钮**：点击页面右上角的"中"或"EN"按钮即可切换语言
- **即时切换**：切换后，所有页面内容（包括导航、标题、正文等）都会立即更新

- **部分内容差异**：某些页面（如 Careers）只在中文版显示，这是根据实际业务需求设计的

- 这是一个产品真伪验证系统，用户可以输入产品代码来验证购买的产品是否为正品
- 目前前端界面已经完成，后端接口已预留，等待后端团队接入实际的验证逻辑

**如何使用**：
- **桌面端**：点击页面右上角的"Verify Product"按钮
- **移动端**：点击右上角的菜单按钮（三条横线），在侧边栏中找到"Verify Product"选项

**技术说明**：
- 前端会发送产品代码到后端 API
- 后端验证后会返回验证结果
- 目前显示的是测试界面，实际验证功能需要等待后端接口完成

### 3. 博客内容管理 (Contentful)

网站的博客文章（Insights）是通过 **Contentful** 内容管理系统来管理的。

#### 如何登录 Contentful 后台

**第一步：获取管理员权限**

要访问 Contentful 后台，你需要先被添加为管理员。请提供你的**邮箱地址**，我们会邀请你成为 Contentful Space 的管理员。

收到邀请邮件后：
1. 点击邮件中的邀请链接
2. 创建 Contentful 账号（如果还没有的话）
3. 登录后即可访问后台

**第二步：登录后台**

1. 访问 [Contentful 官网](https://www.contentful.com/)
2. 使用你的账号登录
3. 选择对应的 Space（空间）：`wbpsfgfg62v2`

![Contentful 内容管理界面](docs/images/contentful-dashboard.png)

#### 如何添加一篇新的博客文章

1. **进入内容管理界面**：
   - 登录后，点击左侧菜单的 "Content"
   - 选择 "blogPost" 内容类型

2. **创建新文章**：
   - 点击右上角的 "Add entry" 按钮
   - 选择 "blogPost" 类型

3. **填写文章信息**：
   - **Title**：文章标题（目前测试内容显示为 "Title"，需要替换为实际标题）
   - **Cover Image**：上传或选择封面图片
   - **Publish DateTime**：发布日期和时间
   - **Category**：文章分类（如 Fashion、Lifestyle、Luxury 等）
   - **Slug**：文章 URL 标识符（建议使用英文，如 `my-article-title`）
   - **Content**：文章正文内容（使用 Rich Text 编辑器）
   - **Excerpt**：文章摘要（可选，如果不填会自动从 Content 中提取前 150 个字符）

4. **发布文章**：
   - 填写完成后，点击右上角的 "Publish" 按钮
   - 文章会立即在网站上显示（可能需要几秒钟的缓存更新）

#### 内容类型说明

- **Title**：目前显示为测试内容 "Title"，需要替换为实际文章标题
- **Content**：使用 Rich Text 格式，支持富文本编辑（加粗、斜体、列表等）
- **多语言支持**：目前使用单一字段，未来可以扩展为多语言字段（titleEn/titleZh）

#### 注意事项

- 文章必须**发布**后才会在网站上显示
- 封面图片需要先上传到 Contentful 的 Assets 中
- 如果文章没有封面图片，网站会使用默认图片
- Content 字段使用 Rich Text 格式，网站会自动将其转换为纯文本显示

## 🎯 各页面功能说明

### 首页 (Home)

首页是网站的门面，包含以下主要部分：

- **Hero Section**：大标题和三个悬浮球动画
- **数据统计**：展示公司核心数据（品牌数量、服务国家、GMV 等）
- **工作流程**：展示我们的 4 步工作流程
- **为什么选择中国**：说明中国市场的重要性
- **合作品牌展示**：滚动展示我们的合作伙伴 Logo
- **成功案例**：展示一些成功案例
- **最新洞察**：显示最新的 3 篇博客文章（固定高度，内容会被裁剪）
- **行动号召**：引导用户联系我们的 CTA 区域

### 关于我们 (About Us)

- 展示公司的使命和愿景
- 介绍我们的核心价值观
- 展示团队文化和工作环境
- 包含公司发展历程

### 服务页面 (Services)

- 展示三大核心服务
- 每个服务都有详细的子页面：
  - **全渠道分销**：展示分销能力和渠道资源
  - **电商运营**：展示电商平台运营能力
  - **营销推广**：展示营销策略和案例
- "Our Process" 部分展示服务流程（2x2 网格布局）

### 合作品牌 (Brands)

- 展示所有合作品牌的 Logo
- 按类别分类（数码配件、包袋鞋履、家居生活等）
- 品牌 Logo 以滚动动画形式展示

### 行业洞察 (Insights)

- **首页预览**：显示最新的 3 篇文章，卡片有固定高度，内容会被裁剪
- **完整页面**：显示所有文章，内容完整展示，无高度限制
- 文章卡片包含：封面图、分类标签、发布日期、标题、摘要
- 点击文章卡片不会跳转（已移除详情页功能）

### 联系我们 (Contact)

- 包含公司地址和联系方式
- 集成 Tally 表单，用户可以直接提交咨询
- 显示三个办公室地址（深圳、上海、香港）

### 产品验证 (Authentication)

- 产品真伪验证界面
- 用户输入产品代码进行验证
- 后端接口已预留，等待接入实际验证逻辑

## 🛠️ 技术特性

### 响应式设计

- 网站完全响应式，在手机、平板、电脑上都有良好的显示效果
- 移动端菜单采用侧边栏设计，体验流畅

### 性能优化

- 图片懒加载：图片在需要显示时才会加载
- 代码分割：不同页面的代码分开加载，提高首屏速度
- 动画优化：使用 CSS 动画和 GPU 加速，确保流畅

### SEO 友好

- 所有页面都有合适的标题和描述
- 语义化 HTML 结构
- 支持多语言，有利于国际化 SEO

按照上面的 Contentful 操作步骤即可。

1. 准备新的 Logo 图片（建议 PNG 格式，透明背景）

## 💡 常见问题

### Q: 为什么博客文章显示的是测试内容？

A: 目前 Contentful 中的文章标题显示为 "Title"，这是测试数据。需要登录 Contentful 后台，编辑文章，将标题替换为实际内容。

### Q: 表单提交后在哪里查看？

A: 使用 Tally 的分享链接（Share URL）即可查看所有提交的数据。具体操作见上面的"表单填写功能"部分。

### Q: 如何修改网站颜色和样式？

A: 网站使用 CSS 变量来管理颜色和样式，可以在 `src/assets/styles/variables.css` 中修改。

### Q: 移动端看不到 Verify Product 按钮？

A: 在移动端，点击右上角的菜单按钮（三条横线），在侧边栏中可以找到 "Verify Product" 选项。
