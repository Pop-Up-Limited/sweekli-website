## 问题回答

### Q1: 服务器选择问题

**问题**：中文网站需要备案（微信内访问），中英文网站是否需要分开托管？

**回答**：

**推荐方案：单服务器架构**

使用腾讯云境内服务器，中英文网站统一托管在同一服务器上。

**优点**：
- 可以完成备案，支持微信内访问
- 统一管理，成本较低
- 域名合并，SEO友好
- 通过CDN可以优化境外访问速度

**建议配置**：
- **服务器**：腾讯云CVM
  - 基础配置：2核4G内存，50GB SSD硬盘
  - 根据实际流量可随时升级
- **CDN**：腾讯云CDN（用于加速境外访问）
- **域名**：统一使用 www.sweekli.com，通过路径区分中英文（/zh/ 前缀或通过语言切换）

**不推荐双服务器架构**（境内+境外），因为：
- 管理复杂，成本较高
- 域名分离，SEO不友好
- 除非有特殊需求，否则单服务器+CDN方案更优

**如果Tally不可用，替代方案**：

**方案一：自建表单（推荐）**
- 使用后端API接收表单数据
- 发送邮件到 zshi@sweekli.com
- **优点**：
  - 完全可控，无访问限制
  - 可以自定义样式和验证
  - 数据安全
- **实现**：
  - 前端：Vue表单组件
  - 后端：Node.js/Python API
  - 邮件服务：SendGrid / AWS SES / 腾讯云邮件服务

**方案二：国内表单服务**
- **金数据（Jinshuju）**：https://jinshuju.net
  - 支持嵌入表单
  - 国内访问稳定
  - 有免费和付费版本
- **问卷星（Wenjuanxing）**：https://www.wjx.cn
  - 支持嵌入表单
  - 国内访问稳定
- **腾讯问卷**：https://wj.qq.com
  - 腾讯旗下，访问稳定
  - 支持嵌入

**方案三：使用现有表单链接**
- 如果Tally不可用，可以暂时使用现有链接：
  - 销售合作：https://cn.sweekli.co/pages/sales-application
  - 市场合作：https://cn.sweekli.co/pages/marketing-application
- 在新窗口中打开，而不是嵌入

**建议**：
1. 先测试Tally在国内的可用性
2. 如果可用，继续使用Tally
3. 如果不可用，推荐使用**方案一（自建表单）**或**方案二（金数据）**

## 技术说明

### 图片路径

- 所有图片使用 getImagePath() 工具函数处理
- 支持开发和生产环境的不同路径
- **位置**：src/utils/imagePath.ts

### 国际化（i18n）

- 使用 vue-i18n 实现中英文切换
- 翻译文件：
  - src/i18n/en.ts - 英文翻译
  - src/i18n/zh.ts - 中文翻译
- 语言切换：通过 locale.value 控制

### Contentful集成

- **配置文件**：src/config/contentful.ts
- **API工具**：src/utils/contentful-api.ts
- **内容类型**：
  - blogPost - 博客文章（Insights页面）
  - jobPosting - 职位信息（Careers页面）

**职位信息更新方法**：
1. 登录Contentful后台
2. 进入 jobPosting Content Type
3. 创建或编辑职位条目
4. 填写所有必填字段（title, department, location, type, description, requirements等）
5. **重要**：点击"Publish"按钮发布条目（仅保存不会显示在网站上）
6. 职位会自动显示在Careers页面

### 响应式设计

- 使用CSS变量和媒体查询
- 移动端优先设计
- 断点：640px, 768px, 1024px
